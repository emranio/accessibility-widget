How to test each tool cheat sheet

  1. Open the floating accessibility button (bottom-right).
  2. Readable font / size / spacing / line height → watch the Typography section reflow.
  3. Contrast · saturation · monochrome → watch the Color swatches and the SVG image.
  4. Highlight links / title → links and headings get outlined.
  5. Reading guide → move the mouse over the long paragraph.
  6. Big cursor → hover the form controls.
  7. Stop animations → the hero gradient, spinner, pulse badge and marquee freeze.
  8. Disable animated images → the GIF becomes a static “GIF Disabled” placeholder.
  9. Screen reader → hover text to hear it (where supported).
  10. Reset → restores everything.

How this page embeds the widget

<!-- built from this repo: npm install && npm run build -->
<script
  src="../dist/accessibility-widget.umd.js"
  data-accessibility-widget-position="bottom-right"
  data-accessibility-widget-offset="24,24"
  data-accessibility-widget-size="58"></script>

Positions: bottom-left (default), bottom-right, top-left, top-right, center-left, center-right, bottom-center, top-center.
